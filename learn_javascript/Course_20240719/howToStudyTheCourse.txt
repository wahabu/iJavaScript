#How To Study The Course
	- Focus
		・ اعزل المشاكل
		・ خالي من أي مواعيد سواء مع الأهل اوالأصدقاء أو أي مهام مع جهة معينة
		・ حاول تكون في وقت من السلام النفسي
		・ وقت ومكان يعزز التركيز وخالي من أي مشتات غالبا اشغل كلتا يديك مع التعلم بحيث تشاهد وتسمع ثم تكرر مع الكتابة ثم كتابة ملاحظات قدر الإمكان
	- Always Search
		・ البحث سر النجاح
		・ مثلا اكتب كلمة map with JavaScript
		・ سوف تتعلم كيف تبحث مع الوقت
		・ ابحث دائما وزود معلوماتك
		・ البحث شيء ممتاز ويوفر الوقت ويضيف لك الكثير ليس عيبا ان تبحث
	- Ask How To Fix Not Fix
		・ المشاكل هي لغة العصر، حل المشكلة بنفسك ولاتبحث عن الحل
		・ أسأل الأخبر منك ماذا اتعلم لحل المشكلة بنفسي
			- سوف يخبرك انت ناقصك واحد واثنين وثلاثة لاجل حل المشكلة
		・ لاتأخذ منه أي تلميحات أو مساعدات في الحل حول بنفسك وستتعلم أشياء كثيرة وانت تحاول
	- Start Idea
		・ لابد أن يكون لديك فكرة من أجل الحصول على دافع للأشياء التي تود تعلمها
		・ عندما تتعلم أكثر سوف ينشئ لديك افكار أكثر لانك اكتسبت مهارات اكثر تفاعلية مع الآخر
		・
	- Check Lessons Code [ https://elzero.org/category/courses/js-course ]
		・ بعد كل درس سوف تجد الكود لكل فيديو
		・ معلومات اضافية، مراجع
		・ كل درس لديه صفحة في الموقع مع كل المعلومات المتعلقة فيه
	- Do The Assignments(التكليفات) [ https://elzero.org/category/assignments/js-assignments/ ]
		・ اهم شيء التكليفات 
		・ بعد كل مجموعة مقاطع سيكون هناك تكليفات لاثبات مافهمته
		・
	- Submit On Group [ https://www.facebook.com/roups/ElzeroStudy ]
		・ لاحقا ربما يكون هناك مجموعة على التلقرام
		・
		・
	- Problems Solving
		・ Sites [ leetcode.com, edabit.com, HackerRank ]
		・ You can write [ JavaScript Problems Solving Wibsite ]
		・ بعد كل مجموعة تكليفات سيكون معهم رابط مسائل للمحاولة في حلها وفحص قدرة الفهم
		・ ويمكنك كذلك الدخول على موقع ترغبه وتحاول حل المشاكل الموجودة واختبار كونك تستطيع أو لا حلها
		・ تذكر حل المشكلات سيبقى معك صديق الرحلة
